1. Hello, World!

One of the first things a programmer learns to do is print "Hello, World!" to the console. In Lua, you would use print("text") to print the word "text" to the console. Try it out below by pressing the execute button!

print("text")

As you can see it worked exactly as we planned it to do. To get "Hello, World!" printed to the console we need to change the text inside the quotation marks. Do this then press execute.

The quotation marks around the text is required as it tells Lua that it is a "string" or text. Try removing the quotation marks and the text and replace it with a number.

Now try adding text next to the number. You would see that lua throws and error into the console. To fix this you can add quotation marks around the entire text.

2. Basic Arithmetic

You can use Lua to do math by typing an equation. The basic operators are "+" for addition, "-" for subtraction, "*" for multiplication" and "/" for division. An example would be "1 + 1". You can then output this to the console by placing it in a print function which looks like, "print(1 + 1)". Type this into the editor and execute it!

print("text")

Now try changing the numbers and the operation.

You can also chain operations such as "1 + 1 * 2" and it follows PEMDAS, so you would get 3 instead of 4. There are more operators such as "^" for exponents and "%" to get the remainder after dividing.
